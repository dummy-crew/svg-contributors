# svg-contributors

Generate contributors svg from github repository

## Installation

```bash
npm i svg-contributors -g
```

## Usage

```bash
dummysvg -o vuejs -n vue

# output: contributors.svg
```

## Options

```
-o, --owner Repository owner
-n, --name Repository name
```
